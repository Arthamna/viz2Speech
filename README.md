How to Run :

```
# compile
npm install expo

npx expo run:android

# if u want spesify target name

npx expo run:android --device {{device_name}}

npx expo start
```

If u have android studio and android device :

On same terminal, press `a` 

Prereq :

use npm install

- expo
- eas


Build app command :

```cmd
eas build -p android --profile preview
```
